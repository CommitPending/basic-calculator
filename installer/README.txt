Calculator Application - Windows Installer
==========================================

Thank you for installing Calculator Application!

GETTING STARTED
---------------
1. Launch the application using the desktop shortcut or Start Menu
2. The backend server will start automatically
3. Your default browser will open to http://localhost:8080
4. Start calculating!

SYSTEM REQUIREMENTS
-------------------
- Windows 10 or later (64-bit)
- Java 23 or higher (bundled if not installed)
- Modern web browser (Chrome, Firefox, Edge)
- 512 MB RAM minimum
- 100 MB disk space

FEATURES
--------
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Keyboard input support
- Dark/Light mode toggle
- Error handling (division by zero)
- Modern, responsive UI

KEYBOARD SHORTCUTS
------------------
- Numbers: 0-9
- Operators: +, -, *, /
- Equals: Enter or =
- Clear: Escape or C
- Backspace: Backspace

TROUBLESHOOTING
---------------
If the application doesn't start:
1. Ensure port 8080 is not in use by another application
2. Check if Java is properly installed (java -version in command prompt)
3. Run as administrator if permission issues occur

To stop the application:
- Use the "Stop Calculator" shortcut in the Start Menu
- Or close the backend console window

UNINSTALL
---------
Use the Windows "Add or Remove Programs" feature or the uninstaller in the Start Menu.
